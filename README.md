# Project Overview
StockSight is a stock market analytics and forecasting web application. It visualizes historical stock data, moving averages, and forecasts future prices using a trained Long Short-Term Memory neural network.

# Setup
1. Clone the repository.
2. Create and activate a Python virtual environment.
3. Install the required dependencies using pip install -r requirements.txt.

# Training the Model
1. Open a terminal in the project directory.
2. Run the training script with python train.py.
3. Wait for the model to download data, preprocess it, and finish training for 50 epochs.
4. Verify that the stock_predictions_model.keras file has been saved in the model directory.

# Running the App
1. Ensure the model has been trained and saved.
2. Open a terminal in the project directory.
3. Start the Streamlit application using streamlit run app.py.
4. Interact with the application in the web browser by specifying a ticker and date range.

# Model Architecture
The forecasting model is built using a 4-layer stacked LSTM architecture to capture complex patterns in time-series data. The network consists of four sequential LSTM layers with 50, 60, 80, and 120 units respectively. Each LSTM layer is followed by a Dropout layer with rates of 0.2, 0.3, 0.4, and 0.5 to mitigate overfitting. A final Dense layer with a single unit outputs the continuous numerical predicted price. The model is compiled using the Adam optimizer and minimizes the mean squared error loss. The training leverages an 80 percent portion of the data processed into 100-day sliding windows using MinMaxScaler for efficient learning.
