# StockSight

# Overview
This is a stock market analytics and forecasting web application. It fetches historical stock data via yFinance, calculates moving averages, and forecasts future prices using a Long Short-Term Memory neural network.

# Tech Stack
1. Python
2. Streamlit
3. yFinance
4. Pandas
5. NumPy
6. Matplotlib
7. scikit-learn
8. TensorFlow
9. Keras

# Setup
1. Clone the repository.
2. Create and activate a Python virtual environment.
3. Install dependencies with pip install -r requirements.txt.

# Training the Model
1. Run the training script via python train.py.
2. The script downloads historical data, preprocesses it, and trains the LSTM model over 50 epochs.
3. The trained model is saved to model/stock_predictions_model.keras.

# Running the App
1. Ensure the model has been trained and the file exists.
2. Start the application with streamlit run app.py.
3. Specify a stock ticker and date range in the browser interface.

# Model Architecture
The forecasting model employs a 4-layer stacked LSTM architecture with 50, 60, 80, and 120 units in sequence. Each LSTM layer is followed by a Dropout layer with rates of 0.2, 0.3, 0.4, and 0.5. A Dense output layer with a single unit produces the prediction. The network is compiled using the Adam optimizer and minimizes mean squared error loss. The input data utilizes a 100-day sliding window, scaled via MinMaxScaler based on an 80/20 split.
